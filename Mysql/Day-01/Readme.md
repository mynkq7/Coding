## Summary
Day 01 introduces core **MySQL fundamentals**, including:
- working with databases and tables  
- defining table structure using data types  
- applying constraints to ensure data integrity  
- selecting and viewing data from tables  
- renaming and altering existing tables  
- inserting single and multiple records into tables  

These concepts form the foundation for relational database design, data storage, and backend development using MySQL.
