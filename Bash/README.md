# Day 01 — Bash Basics

## Topics Covered
- Directory listing (`ls`)
- Navigating directories (`cd`, `pwd`)
- Printing output (`echo`)
- Viewing file contents (`cat`)
- File searching with pipes (`grep`)
- Copying files and directories (`cp`)
- Moving and renaming files (`mv`)
- Removing files and directories (`rm`, `rmdir`)
- Creating files and directories (`touch`, `mkdir`)
- Viewing manual pages (`man`)

## File in this folder
- `day-01.sh` — main script

## Summary
Day 01 introduces essential **Linux/Bash command-line fundamentals** such as:
- listing and navigating directories  
- viewing and manipulating files  
- copying, moving, and deleting files/directories  
- using command options and flags  
- chaining commands using pipes  
- accessing built-in documentation with `man`  

These commands form the foundation for working efficiently in Linux environments, servers, and development workflows.
