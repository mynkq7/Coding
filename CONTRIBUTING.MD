# Contributing Guidelines

## Commit Message Style
- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation changes
- `chore:` cleanup or minor edits

## Folder Naming
- Use Day-01, Day-02, etc.
- Use lowercase for files: `guessing_game.py`

## Code Style
- Add comments
- Keep files small and readable
- Follow Python best practices (PEP8)
